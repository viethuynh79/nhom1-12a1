## 🎨 Tùy Chỉnh

### **1. Đổi Màu Sắc**

Mở `css/style.css`, tìm `:root` và thay đổi:

```css
:root {
    /* Đổi màu chính */
    --primary-color: #667eea;      /* Tím xanh */
    --secondary-color: #764ba2;    /* Tím */
    --accent-color: #f093fb;       /* Hồng */
    
    /* Hoặc dùng màu của bạn */
    --primary-color: #FF5733;
    --secondary-color: #C70039;
    --accent-color: #900C3F;
}
```

### **2. Đổi Font Chữ**

Thay link Google Fonts trong `<head>` của `index.html`:

```html
<!-- Font hiện tại: Inter + Poppins -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&family=Poppins:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">

<!-- Thay bằng font khác (ví dụ: Roboto) -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700;900&display=swap" rel="stylesheet">
```

Sau đó update CSS:

```css
:root {
    --font-primary: 'Roboto', sans-serif;
    --font-display: 'Roboto', sans-serif;
}
```

### **3. Tắt Animations (Nếu Cần)**

Thêm vào cuối `css/style.css`:

```css
/* Tắt tất cả animations */
* {
    animation: none !important;
    transition: none !important;
}
```

### **4. Thay Đổi Nội Dung**

#### **a) Cập nhật thông tin members:**
Mở `index.html`, tìm class `member-card` và sửa:
- Tên thành viên
- Mô tả
- Điểm mạnh
- Social media links

#### **b) Cập nhật translations:**
Mở `js/main.js`, tìm object `translations` và sửa:
```javascript
const translations = {
    vi: {
        'member1-name': 'Tên Mới',
        'member1-description': 'Mô tả mới...'
    },
    en: {
        'member1-name': 'New Name',
        'member1-description': 'New description...'
    }
};
```

**✅ Ưu điểm:**
- Miễn phí vĩnh viễn
- Deploy trong 2 phút (drag & drop)

- HTTPS tự động
- CDN toàn cầu

**📝 Các bước:**

2. Sign up (miễn phí)
3. Vào "Sites" → Kéo thả folder vào
4. Đợi 30 giây → ✅ **Live!**
5. Đổi tên site: Settings → Change site name

### **Phương Pháp 2: GitHub Pages (Stable ⭐⭐⭐⭐)**

**✅ Ưu điểm:**
- Miễn phí vĩnh viễn
- Tin cậy (GitHub hosting)
- Domain: `username.github.io/repo-name`

**📝 Các bước:**
1. Tạo repo mới trên GitHub
2. Upload files lên
3. Settings → Pages → Source: main branch
4. Đợi 2-5 phút → ✅ **Live!**

### **Phương Pháp 3: Vercel (Fast ⭐⭐⭐⭐⭐)**

**✅ Ưu điểm:**
- Siêu nhanh (Edge network)
- Free tier generous
- Domain: `project.vercel.app`

**📝 Các bước:**
1. Truy cập: https://vercel.com/
2. Sign up với GitHub
3. Import repository
4. Deploy → ✅ **Live!**

Nếu muốn domain riêng (ví dụ: `nhom1-lequydon.com`):

**🇻🇳 Mua domain tại:**
- **MATBAO** (https://matbao.net/) - 150,000đ/năm
- **PA Vietnam** (https://pavietnam.vn/) - 200,000đ/năm
- **TENTEN** (https://tenten.vn/) - 180,000đ/năm

**🌍 Hoặc quốc tế:**
- **Namecheap** (https://namecheap.com/) - $8.88/năm

**Sau khi mua domain:**
1. Vào Netlify/Vercel dashboard
2. Domain settings → Add custom domain
3. Nhập domain của bạn
4. Cấu hình DNS theo hướng dẫn
5. Đợi 24-48h → ✅ **Live với domain riêng!**
